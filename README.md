# 🌾 KrishiMitra - Smart Farming Dashboard

KrishiMitra is an intelligent farming assistant that empowers farmers to make data-driven decisions through crop prediction, plant disease detection, and vendor recommendations.

## Problem Statement
Farmers face critical challenges in modern agriculture:
- Uncertainty in crop selection due to varying soil conditions
- Lack of early disease detection leading to crop losses
- Limited access to market prices and vendor information
- Difficulty in accessing expert agricultural advice

## Solution
KrishiMitra provides an integrated smart farming platform with:
- **Intelligent Crop Prediction**: ML-based recommendations based on soil parameters
- **Disease Detection**: Computer vision-enabled leaf disease identification
- **Pesticide Recommendations**: Targeted solutions for identified diseases
- **Market Insights**: Real-time crop price trends and forecasts
- **Vendor Locator**: Find nearby agro-vendors with discounts

## Features
✅ Crop prediction dashboard with real-time insights  
✅ Camera-based disease detection using image analysis

## Installation & Setup

### Prerequisites
- Python 3.8 or higher
- Modern web browser
- Git

### Quick Start
1. **Clone repository**
   ```bash
   git clone https://github.com/yourusername/krishimitra.git
   cd krishimitra
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Open dashboard**
   - Open `app/index.html` in your browser
   - Or run: `python -m http.server 8000`

## Usage Guide

### Crop Prediction
1. Enter soil parameters (Nitrogen, Phosphorus, Rainfall)
2. Click "Predict Crop"
3. View recommendations and market prices

### Disease Detection
1. Click camera button (bottom-right)
2. Upload leaf image
3. Get disease diagnosis and pesticide recommendations

### Price Trends
View historical crop prices and plan harvest timing

### Vendor Finder
Browse nearby agro-vendors with contact information and discounts
